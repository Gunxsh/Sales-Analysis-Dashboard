# Sales Performance Analysis using SQL

## Project Overview

This project analyzes sales performance using SQL, Python, and Power BI. The objective is to identify sales trends, product performance, regional performance, and provide business recommendations based on data analysis.

  - Quantity
  - Discount
  - Profit

---

## Business Questions

This project answers the following business questions:

1. What is the total sales?
2. What is the total profit?
3. Which product category generates the highest sales?
4. Which category generates the highest profit?
